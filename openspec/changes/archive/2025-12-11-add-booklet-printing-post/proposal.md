# ブログ記事「Ubuntu での小冊子印刷」の追加

## 目的
`pdfbook2` を使用して Ubuntu で小冊子印刷を行う方法を共有するため、新しいブログ記事「完全爆弾解除マニュアルのマニュアルPDFをいい感じに印刷する、または Ubuntu で小冊子印刷をする方法」を追加します。

## 要件
記事は以下を網羅する必要があります：
1.  **TL;DR**: `pdfbook2` のインストールと使用方法の簡潔な説明。
2.  **背景**: 「完全爆弾解除マニュアル」を印刷する必要があったこと。
3.  **試行錯誤**: うまくいかなかった他のツール（`mkbookpdf`, `boomaga`）への言及。
4.  **解決策**: `pdfbook2` のインストール方法（`texlive-extra-utils`）と詳細な手順。
5.  **環境**: Ubuntu 24.04.3 LTS.

## アプローチ
- `content-org/all-posts.org` 内のドラフトをリライトします。
- ステータスを `DRAFT` から `DONE` に変更します。
- 適切なタグとメタデータを設定します。
